Project Overview
This project simulates the diffraction pattern of light passing through a cross-shaped slit. The simulation incorporates Fresnel diffraction effects to visualize real physical phenomena and includes dynamic changes over time. The visualization demonstrates how light waves interfere and diffract, creating complex patterns similar to those observed in real optical experiments.
Developed while [Intro to Creative Computing] course
Physical Wave Theory
Basic Diffraction Physics
In optics, diffraction occurs when light waves encounter obstacles or openings. The fundamental equation for single-slit diffraction intensity is:
I(θ) = I₀ * (sin(α)/α)² where α = (πa/λ)sin(θ)
I₀: Maximum intensity
a: Slit width
λ: Wavelength
θ: Angle from the central maximum
In my code, I implement this diffraction pattern through:
float slitX = sin(dx * PI) / (dx * PI); float slitY = sin(dy * PI) / (dy * PI); float crossPattern = (slitX * slitX + slitY * slitY);
Fresnel Diffraction
Fresnel diffraction occurs in the near-field region where the wavefront curvature cannot be ignored. The pattern depends on the distance from the source in a quadratic manner. I implement this through:
float r = sqrt(dx*dx + dy*dy); float fresnelEffect = cos(r * r * PI + time) * 0.5 + 0.5; // Final intensity calculation float intensity = crossPattern * fresnelEffect; intensity = pow(intensity, 1.5); // Contrast enhancement intensity *= exp(-r * 0.5); // Distance attenuation
This implementation creates a dynamic diffraction pattern where:

-> The cross pattern represents the basic single-slit diffraction
-> The Fresnel effect adds near-field interference patterns
-> Distance attenuation models realistic intensity decay
-> Time-dependent phase creates subtle pattern evolution
Visualization Through Color Mapping
The diffraction patterns are visualized using a red-dominated color scheme to simulate monochromatic laser light:
float r = intensityMap[x][y]; float g = intensityMap[x][y] * 0.2; float b = intensityMap[x][y] * 0.2; pixels[y * width + x] = color(r, g, b);
This color mapping creates an intuitive visualization where:

-> Bright red areas indicate constructive interference
-> Dark areas indicate destructive interference
-> The intensity gradient shows the continuous nature of the diffraction pattern

Additionally, the intensityMap array provides temporal smoothing:
intensityMap[x][y] = lerp(intensityMap[x][y], intensity, 1);
Parameter Effects and Analysis
The diffraction pattern is controlled by several key parameters that can be adjusted to achieve different visual effects:
The scaling factor in dx and dy calculations determines the spatial frequency of the pattern. Larger values create finer diffraction patterns, while smaller values create broader patterns.
float dx = (x - centerX) / 100.0; float dy = (y - centerY) / 100.0;
The time-dependent phase modulation creates dynamic pattern changes. This adds subtle breathing animation to the pattern, making it appear more organic and alive.
float phase = sin(time * 0.1) * 0.1; intensity *= 2 + phase;
Pattern contrast and decay are controlled by two main factors that can be adjusted to match different experimental conditions:

-> Contrast enhancement using power function
-> Exponential decay with distance
intensity = pow(intensity, 1.5); // Contrast intensity *= exp(-r * 0.5); // Decay
Full code:
float[][] intensityMap; float time = 0; void setup() { size(800, 800); colorMode(RGB, 1.0); intensityMap = new float[width][height]; } void draw() { loadPixels(); // 중심점 float centerX = width/2; float centerY = height/2; // 회절 패턴 계산 for (int x = 0; x < width; x++) { for (int y = 0; y < height; y++) { float dx = (x - centerX) / 100.0; float dy = (y - centerY) / 100.0; // 십자가 모양의 슬릿에 대한 회절 패턴 float intensity = calcDiffraction(dx, dy); // 시간에 따른 약간의 변화 float phase = sin(time * 0.1) * 0.1; intensity *= 2 + phase; // 강도 맵 업데이트 intensityMap[x][y] = lerp(intensityMap[x][y], intensity, 1); // 색상 설정 (빨간색 계열) float r = intensityMap[x][y]; float g = intensityMap[x][y] * 0.2; float b = intensityMap[x][y] * 0.2; pixels[y * width + x] = color(r, g, b); } } updatePixels(); time += 0.1; } float calcDiffraction(float dx, float dy) { // 기본 십자가 패턴 float slitX = sin(dx * PI) / (dx * PI); float slitY = sin(dy * PI) / (dy * PI); // NaN 처리 if (Float.isNaN(slitX)) slitX = 1; if (Float.isNaN(slitY)) slitY = 1; // 십자가 패턴 생성 float crossPattern = (slitX * slitX + slitY * slitY) ; // 프레넬 회절 효과 추가 float r = sqrt(dx*dx + dy*dy); float fresnelEffect = cos(r * r * PI + time) * 0.5 + 0.5; // 최종 강도 계산 float intensity = crossPattern * fresnelEffect; // 강도 조정 intensity = pow(intensity, 1.5); // 대비 증가 intensity *= exp(-r * 0.5); // 거리에 따른 감쇠 return constrain(intensity, 0, 1); } void mousePressed() { for (int x = 0; x < width; x++) { for (int y = 0; y < height; y++) { intensityMap[x][y] = 0; } } }
