Fluid For Fun
Overview
This project uses Processing to implement a real-time fluid simulation and visualize it in the style of an ink drawing. It simulates fluid dynamics based on the Navier-Stokes equations.
Basic Fluid Simulation
This is the core function that calculates fluid velocity and density. It uses diffuse() for diffusion, advect() for movement, and project() to maintain incompressibility.
Visual Effects
This function creates marbling effects based on density and curl. It uses HSB color mode to implement smooth color transitions reminiscent of ink wash paintings.
Vortex Pattern
Creates a vortex pattern that rotates around a center point.
Spiral Pattern
Creates a spiral pattern that expands outward from the center while rotating.
