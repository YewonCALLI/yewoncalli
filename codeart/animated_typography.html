Gravity For Fun
What happens when physical phenomena meet typography? This project combines various physics-based effects with typography to create dynamic, interactive art that responds to user input through different natural phenomena like magnetic fields, wave propagation, and quantum effects.
Project Overview
This project implements real-time fluid simulation using Processing and visualizes it in an ink wash painting style. It simulates fluid dynamics based on the Navier-Stokes equations and allows users to create various patterns through different physical interactions.
Key Features
Each particle in the system follows physical laws and can be influenced by various forces. The project includes six different physical effects that can be triggered through keyboard inputs, creating unique visual patterns that mimic natural phenomena.
Technical Approach
The project is built using Processing, implementing particle systems and physics-based animations. Each effect is carefully designed to mimic real physical phenomena while maintaining artistic quality.
1. Ripple Effect
Creates concentric waves that propagate from the center point, simulating water ripples. The wave amplitude decreases with distance, creating natural-looking wave patterns.
PVector center = new PVector(width/2, height/2); float distance = PVector.dist(position, center); float ripple = sin(distance * 0.05 - frameCount * 0.1) * 2; PVector dir = PVector.sub(position, center).normalize(); applyForce(dir.mult(ripple));
Ripple Effect Visualization
2. Magnetic Field Effect
Simulates magnetic field lines by applying perpendicular forces to particles. The force magnitude decreases with distance from the center, following inverse square law.
PVector toCenter = PVector.sub(center, position); float distance = toCenter.mag(); PVector magnetic = new PVector(toCenter.y, -toCenter.x); magnetic.normalize(); magnetic.mult(200 / distance);
Magnetic Field Effect
3. DNA Double Helix
Creates a spiral pattern using sine and cosine functions with different phases for each particle, resulting in a DNA-like double helix structure.
float y = original.y + sin(frameCount * frequency + phase) * amplitude * 3; float x = original.x + cos(frameCount * frequency + phase) * amplitude; PVector target = new PVector(x, y);
DNA Helix Animation
4. Resonance
Creates complex resonance patterns by combining multiple sine waves. The particles appear to resonate as position-based and time-based oscillations combine.
float resonance = sin(frameCount * frequency) * sin(position.x * 0.02) * sin(position.y * 0.02); applyForce(new PVector(resonance, resonance));
Resonance Pattern
5. Quantum Tunneling
Visualizes quantum tunneling phenomena by allowing particles to occasionally "jump" to different positions, creating a unique visual effect reminiscent of quantum mechanics.
if (random(1) < 0.01) { position.x += random(-50, 50); position.y += random(-50, 50); }
Quantum Tunneling Effect
6. Sound Visualization
Uses Perlin noise to generate natural wave patterns. Each particle moves according to a noise field, creating patterns similar to sound wave propagation.
float noiseVal = noise(position.x * 0.01, position.y * 0.01, frameCount * 0.02); float angle = noiseVal * TWO_PI * 2; PVector noiseForce = PVector.fromAngle(angle); noiseForce.mult(2); applyForce(noiseForce);
